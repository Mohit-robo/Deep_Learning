# Activation Functions

This subdirectory contains implementations of various activation functions used in neural networks.

## Available Functions

- **Sigmoid**: Logistic function that outputs values between 0 and 1.
- **Tanh**: Hyperbolic tangent function that outputs values between -1 and 1.
- **ReLU**: Rectified Linear Unit that outputs max(0, x).
- **Softmax**: Converts logits to probability distributions.

Each activation function is implemented in `act_function.py` with PyTorch-style forward and backward passes.
